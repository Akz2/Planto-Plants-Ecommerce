# 🌿 Planto – Plants E-Commerce Website

A premium plants e-commerce website built with **Next.js**, faithfully replicating a Figma design with full responsiveness, dark theme, glassmorphism UI, and smooth animations.

## 🚀 Live Demo

> Deployed on Vercel — link available after deployment

## ✨ Features

- 🌱 **Hero Section** – Animated plant card carousel with review widget
- 🌿 **Trendy Plants** – Alternating horizontal product cards
- 🏆 **Top Selling** – 3-column responsive product grid
- ⭐ **Customer Reviews** – Organic card shapes with star ratings
- 🍃 **Best O2 Section** – Full-featured plant showcase slider
- 📱 **Fully Responsive** – Mobile, tablet, and desktop layouts
- 🎨 **Dark Theme** – Deep forest green color palette
- ✨ **Glassmorphism** – Blur & transparency effects throughout
- 🔄 **Micro-animations** – Float, hover, scroll animations

## 🛠 Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Styling**: CSS Modules (Vanilla CSS)
- **Images**: Next.js Image optimization
- **Fonts**: Inter (Google Fonts)

## 📁 Project Structure

```
Planto/
├── app/
│   ├── layout.js       # Root layout & metadata
│   ├── page.js         # Main page
│   └── globals.css     # Global styles & design tokens
├── components/
│   ├── Navbar/         # Sticky nav with mobile menu
│   ├── Hero/           # Hero with plant card carousel
│   ├── TrendyPlants/   # Horizontal feature cards
│   ├── TopSelling/     # Product grid section
│   ├── CustomerReviews/# Review cards
│   ├── BestO2/         # Slider showcase
│   ├── SectionTitle/   # Reusable section header
│   └── Footer/         # Footer with links
└── public/
    └── images/         # Plant photos
```

## 🏃 Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## 📦 Build

```bash
npm run build
npm start
```
